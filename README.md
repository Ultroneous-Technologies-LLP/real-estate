# Real Estate Website Landing Page – Property, Housing & Modern Realtor UI

Elegant real estate website landing page design. Perfect for property listings, housing agencies, and realtors showcasing modern properties and apartments.

## 🚀 Features

- Modern UI/UX design tailored for real estate
- Responsive and mobile-friendly
- Built with [Next.js](https://nextjs.org/) (App Router)
- Optimized fonts using [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts)
- SEO-friendly structure
- Easy to customize and extend for agencies, realtors, and property listings
